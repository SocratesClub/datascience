## 课程描述（Course Description）

本课程注重编程训练、数学建模、可计算思维。本课程致力于介绍python编程和数据科学基础知识。

- 时间：周三 第5-6节 逸夫楼C-405 1-17周
- 教师：王成军

[王成军](http://chengjun.github.io)，南京大学新闻传播学院副教授，奥美数据科学实验室主任，南京大学计算传播学实验中心副主任、香港城市大学Web Mining Lab成员。
